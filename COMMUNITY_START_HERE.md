# ALBA Community Start Here

If you want to share ALBA in Discord and let people help improve it, use this pack.

## Files to attach or link

- `ALBA_CONCEPT_UPDATED_IT.docx`
- `ALBA_CONCEPT_UPDATED_EN.docx`
- `public/alba-infographic-it.svg`
- `public/alba-infographic-en.svg`
- public demo URL
- GitHub repository URL

If Discord does not preview SVG files well, share the infographic route links too:

- `/infographic?lang=it`
- `/infographic?lang=en`

## Files to point contributors to

- `CONTRIBUTING.md`
- `RUN_ALBA_LOCALLY.md`

## Files ready to copy and paste

- `DISCORD_POST_IT.md`
- `DISCORD_POST_EN.md`

## Recommended flow

1. Publish the public demo
2. Publish the GitHub repository
3. Post the Discord message
4. Attach concept and infographic
5. Invite feedback and pull requests

## Goal

Keep the community workflow simple:

- Discord for conversation
- GitHub for actual prototype improvements
